## Descrição
Aplicação fullstack para gestão de treinos, exercícios e dietas entre personal trainers, nutricionistas e alunos, com autenticação e persistência em MongoDB.

## Tecnologias
- Next.js 14+ (App Router)
- MongoDB Atlas & Mongoose
- HeroUI (NextUI) & Tailwind CSS
- NextAuth (Auth.js)

## Funcionalidades
- [x] Cadastro e Login de Usuários
- [x] CRUD de Exercícios, Treinos e Dietas
- [x] Atribuição de treinos/dietas a alunos com controle de consumo de água e calorias

## Configuração
Instruções para rodar o projeto localmente:
1. Clone o repositório
2. Configure o arquivo `.env` com a STRING DE CONEXÃO DO MONGO.
3. `npm install`
4. `npm run dev`
